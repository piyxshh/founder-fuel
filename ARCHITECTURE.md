# FounderFuel Architecture & Stack
- **Role**: AI-Driven Content Engine for Founders.
- **Frontend**: Next.js (App Router), Tailwind CSS v4, Framer Motion. (:3000)
- **Backend**: Express.js (Custom Server), Node.js. (:4000)
- **Database**: PostgreSQL (NeonDB) with Prisma ORM.
- **AI**: LangChain.js.
- **Goal**: Learn deep full-stack fundamentals. No "black box" code allowed.
